Propuesta PMV institucional

Sistema de asistencia docente con un estilo visual cercano a Fe y Alegría

Esta primera versión presenta el flujo más elemental del sistema: visualizar la lista de estudiantes, marcar asistencia y registrar observaciones. Además, incorpora un contador de interés y una caja de comentarios para recopilar recomendaciones de los docentes.

¿Qué valida este PMV?

  • Si el registro de asistencia resulta claro para el docente.
  • Si el flujo propuesto ahorra tiempo dentro del aula.
  • Si los profesores muestran interés real en usar la herramienta.
  • Si las observaciones recogidas permiten mejorar la propuesta.

Problema que buscamos resolver

En muchos contextos educativos, el registro de asistencia sigue realizándose de forma manual, lo que implica pérdida de tiempo, interrupciones en la clase y poca agilidad para consultar la información. Este PMV permite validar una versión sencilla de la solución, con una apariencia visual inspirada en la identidad institucional.
