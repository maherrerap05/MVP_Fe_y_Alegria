Propuesta PMV para instituciones Fe y Alegría

Sistema de asistencia docente adaptable para la red educativa Fe y Alegría

Esta versión del PMV permite validar un flujo básico de registro de asistencia por horarios, pensado para ser utilizado en diferentes instituciones de Fe y Alegría. El sistema permite seleccionar un horario de clase, cargar la lista correspondiente de estudiantes, registrar la asistencia, generar un reporte y recoger recomendaciones de los docentes.

¿Qué valida este PMV?

  • Si el registro de asistencia por horarios resulta claro para los docentes.
  • Si el sistema puede adaptarse a diferentes colegios de Fe y Alegría.
  • Si el flujo propuesto ahorra tiempo dentro del aula.
  • Si los reportes y comentarios ayudan a mejorar la solución.

Problema que buscamos resolver

En distintas instituciones educativas, el registro de asistencia puede seguir realizándose de forma manual o poco centralizada. Esto genera pérdida de tiempo, dificultad para consultar información y poca trazabilidad por horarios. Este PMV propone una solución sencilla, adaptable a varios colegios de Fe y Alegría, que permite registrar asistencia por clase y recopilar retroalimentación docente.

1

Seleccionar horario

El docente elige el horario de clase correspondiente.

2

Cargar estudiantes

El sistema muestra la lista asociada a ese horario.

3

Registrar asistencia

El docente marca quién asistió, quién faltó y quién queda sin marcar.

4

Generar reporte

El sistema permite descargar un PDF sencillo del registro realizado.
